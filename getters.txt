<div class="section" id="available-configuration-templates">

<ul class="simple">

<li><code class="code docutils literal notranslate"><span class="pre">delete_probes</span></code> (JunOS, IOS-XR): Removes RPM/SLA probes.</li>

<div class="section" id="optional-arguments">
<h2>Optional arguments<a class="headerlink" href="#optional-arguments" title="Permalink to this headline">¶</a></h2>
<p>NAPALM supports passing certain optional arguments to some drivers. To do that you have to pass a dictionary via the
<code class="code docutils literal notranslate"><span class="pre">optional_args</span></code> parameter when creating the object:</p>
<div class="highlight-default notranslate"><div class="highlight"><pre><span></span><span class="gp">&gt;&gt;&gt; </span><span class="kn">from</span> <span class="nn">napalm</span> <span class="kn">import</span> <span class="n">get_network_driver</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">driver</span> <span class="o">=</span> <span class="n">get_network_driver</span><span class="p">(</span><span class="s1">&#39;eos&#39;</span><span class="p">)</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">optional_args</span> <span class="o">=</span> <span class="p">{</span><span class="s1">&#39;my_optional_arg1&#39;</span><span class="p">:</span> <span class="s1">&#39;my_value1&#39;</span><span class="p">,</span> <span class="s1">&#39;my_optional_arg2&#39;</span><span class="p">:</span> <span class="s1">&#39;my_value2&#39;</span><span class="p">}</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">device</span> <span class="o">=</span> <span class="n">driver</span><span class="p">(</span><span class="s1">&#39;192.168.76.10&#39;</span><span class="p">,</span> <span class="s1">&#39;dbarroso&#39;</span><span class="p">,</span> <span class="s1">&#39;this_is_not_a_secure_password&#39;</span><span class="p">,</span> <span class="n">optional_args</span><span class="o">=</span><span class="n">optional_args</span><span class="p">)</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">device</span><span class="o">.</span><span class="n">open</span><span class="p">()</span>
</pre></div>
</div>
<div class="section" id="list-of-supported-optional-arguments">
<h3>List of supported optional arguments<a class="headerlink" href="#list-of-supported-optional-arguments" title="Permalink to this headline">¶</a></h3>
<ul class="simple">
<li><code class="code docutils literal notranslate"><span class="pre">allow_agent</span></code> (ios, iosxr, nxos_ssh) - Paramiko argument, enable connecting to the SSH agent (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">alt_host_keys</span></code> (ios, iosxr, nxos_ssh) - If <code class="docutils literal notranslate"><span class="pre">True</span></code>, host keys will be loaded from the file specified in <code class="docutils literal notranslate"><span class="pre">alt_key_file</span></code>.</li>
<li><code class="code docutils literal notranslate"><span class="pre">alt_key_file</span></code> (ios, iosxr, nxos_ssh) - SSH host key file to use (if <code class="docutils literal notranslate"><span class="pre">alt_host_keys</span></code> is <code class="docutils literal notranslate"><span class="pre">True</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">auto_probe</span></code> (junos) - A timeout in seconds, for which to probe the device. Probing determines if the device accepts remote connections. If <cite>auto_probe</cite> is set to <code class="docutils literal notranslate"><span class="pre">0</span></code>, no probing will be done. (default: <code class="docutils literal notranslate"><span class="pre">0</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">auto_rollback_on_error</span></code> (ios) - Disable automatic rollback (certain versions of IOS support configure replace, but not rollback on error) (default: <code class="docutils literal notranslate"><span class="pre">True</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">config_lock</span></code> (iosxr_netconf, iosxr, junos) - Lock the config during open() (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">lock_disable</span></code> (junos) - Disable all configuration locking for management by an external system (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">config_private</span></code> (junos) - Configure private, no DB locking (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">canonical_int</span></code> (ios) - Convert operational interface’s returned name to canonical name (fully expanded name) (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">dest_file_system</span></code> (ios) - Destination file system for SCP transfers (default: <code class="docutils literal notranslate"><span class="pre">flash:</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">enable_password</span></code> (eos) - Password required to enter privileged exec (enable) (default: <code class="docutils literal notranslate"><span class="pre">''</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">force_no_enable</span></code> (ios, nxos_ssh) - Do not automatically enter enable-mode on connect (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">global_delay_factor</span></code> (ios, nxos_ssh) - Allow for additional delay in command execution (default: <code class="docutils literal notranslate"><span class="pre">1</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">ignore_warning</span></code> (junos) - Allows to set <cite>ignore_warning</cite> when loading configuration to avoid exceptions via junos-pyez. (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">keepalive</span></code> (iosxr, junos) - SSH keepalive interval, in seconds (default: <code class="docutils literal notranslate"><span class="pre">30</span></code> seconds).</li>
<li><code class="code docutils literal notranslate"><span class="pre">key_file</span></code> (ios, iosxr_netconf, iosxr, junos, nxos_ssh) - Path to a private key file. (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">port</span></code> (eos, ios, iosxr_netconf, iosxr, junos, nxos, nxos_ssh) - Allows you to specify a port other than the default.</li>
<li><code class="code docutils literal notranslate"><span class="pre">secret</span></code> (ios, nxos_ssh) - Password required to enter privileged exec (enable) (default: <code class="docutils literal notranslate"><span class="pre">''</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">ssh_config_file</span></code> (ios, iosxr, junos, nxos_ssh) - File name of OpenSSH configuration file.</li>
<li><code class="code docutils literal notranslate"><span class="pre">ssh_strict</span></code> (ios, iosxr, nxos_ssh) - Automatically reject unknown SSH host keys (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>, which means unknown SSH host keys will be accepted).</li>
<li><code class="code docutils literal notranslate"><span class="pre">ssl_verify</span></code> (nxos) - Requests argument, enable the SSL certificates verification. See requests ssl-cert-verification for valid values (default: <code class="docutils literal notranslate"><span class="pre">None</span></code> equivalent to <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">transport</span></code> (eos, ios, nxos) - Protocol to connect with (see <a class="reference internal" href="#the-transport-argument">The transport argument</a> for more information).</li>
<li><code class="code docutils literal notranslate"><span class="pre">use_keys</span></code> (ios, iosxr, nxos_ssh) - Paramiko argument, enable searching for discoverable private key files in <code class="docutils literal notranslate"><span class="pre">~/.ssh/</span></code> (default: <code class="docutils literal notranslate"><span class="pre">False</span></code>).</li>
<li><code class="code docutils literal notranslate"><span class="pre">eos_autoComplete</span></code> (eos) - Allows to set <cite>autoComplete</cite> when running commands. (default: <code class="docutils literal notranslate"><span class="pre">None</span></code> equivalent to <code class="docutils literal notranslate"><span class="pre">False</span></code>)</li>
<li><code class="code docutils literal notranslate"><span class="pre">config_encoding</span></code> (iosxr_netconf) - Set encoding to either <code class="docutils literal notranslate"><span class="pre">&quot;xml&quot;</span></code> or <code class="docutils literal notranslate"><span class="pre">&quot;cli&quot;</span></code> for configuration load methods. (default: <code class="docutils literal notranslate"><span class="pre">&quot;cli&quot;</span></code>)</li>
<li><code class="code docutils literal notranslate"><span class="pre">eos_fn0039_config</span></code> (eos) - Transform old style configuration to the new style, available beginning with EOS release 4.23.0, as per FN 0039. Beware
that enabling this option will change the configuration you’re loading through NAPALM. Default: <code class="docutils literal notranslate"><span class="pre">False</span></code> (won’t change your configuration commands).
.. versionadded:: 3.0.1</li>
</ul>
</div>
